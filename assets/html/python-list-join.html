Join Two List

Ada beberapa cara untuk menggabungkan dua atau lebih list dengan Python.

Salah satu cara termudah adalah dengan menggunakan operator+.

Contoh

Gabungkan dua list:

list1 = ["a", "b", "c"]
list2 = [1, 2, 3]

list3 = list1 + list2
print (list3)
  • Cobalah!

Cara lain untuk menggabungkan dua list adalah dengan menambahkan semua item dari list2 ke list1, satu per satu:

Contoh

Tambahkan list2 ke list1:

list1 = ["a", "b", "c"]
list2 = [1, 2, 3]

for x in list2:
   list1.append(x)

print (list1)
  • Cobalah!

Atau kalian bisa menggunakan metodeextend(), yang tujuannya adalah menambahkan elemen dari satu list ke list lain:

Contoh

Gunakan metodeextend()untuk menambahkan list2 di akhir list1:

list1 = ["a", "b", "c"]
list2 = [1, 2, 3]

list1.extend(list2)
print (list1)
  • Cobalah!
