Python Variable Assign Multiple Values

Many Values to Multiple Variables

Python memungkinkan kalian untuk menetapkan nilai ke beberapa variabel dalam satu baris:

Contoh

x, y, z = "Orange", "Banana", "Cherry"
print (x)
print (y)
print (z)
  • Cobalah!
Note: Pastikan jumlah variabel sama dengan jumlah nilai (value), kalau tidak maka kalian akan mendapatkan error.

One Value to Multiple Variables

Dan kalian dapat menetapkan nilai yang sama ke beberapa variabel dalam satu baris:

Contoh

x = y = z = "Orange"
print (x)
print (y)
print (z)
  • Cobalah!

Unpack a Collection

Jika kalian memiliki kumpulan nilai dalam list, tuple, dll. Python memungkinkan kalian untuk mengekstrak nilai ke dalam variabel. Hal ini disebut denganunpacking.

Contoh

fruits = ["apple", "banana", "cherry"]
x, y, z = fruits
print (x)
print (y)
print (z)
  • Cobalah!

Pelajari lebih lanjut tentangunpackingdi bagian Unpack Tuples.
