Python Syntax

Execute Python Syntax

Seperti yang sudah kita pelajari di halaman sebelumnya, syntax Python dapat dieksekusi dengan menulis secara langsung di command line:

>>> print ("Hello, World!")
Hello, World!

Atau dengan membuat sebuah file Python di server, menggunakan file extension .py, dan menjalankannya di command line:

C:\Users\ Nama User >python myfile.py

Python Indentation

Indentasi mengacu pada spasi di awal baris kode.

Di mana dalam bahasa pemrograman lain indentasi dalam kode hanya untuk keperluan membaca kode, indentasi pada Python sangat penting.

Python menggunakan indentasi untuk menunjukkan blok kode.

Contoh

if 5 > 2:
  print ("Lima lebih besar dari Dua!")
  • Cobalah!

Python akan memberi pesan error jika kalian tidak menggunakan indentasi:

Contoh

if 5 > 2:
print ("Lima lebih besar dari Dua!")
  • Cobalah!

Jumlah spasi itu terserah kalian para programmer, tetapi setidaknya harus satu spasi

Contoh

if 5 > 2:
  print ("Lima lebih besar dari Dua!")
if 5 > 2:
      print ("Lima lebih besar dari Dua!")
  • Cobalah!

Kalian harus menggunakan jumlah spasi yang sama di dalam blok kode yang sama, kalau tidak Python akan memberikan pesan error:

Contoh

if 5 > 2:
  print ("Lima lebih besar dari Dua!")
      print ("Lima lebih besar dari Dua!")
  • Cobalah!

Python Variables

Dalam Python, variabel dibuat saat kalian memberikan nilai padanya:

Contoh

x = 5
y = "Hello, World!"

Python tidak memiliki perintah untuk mendeklarasikan variabel

Kalian akan belajar lebih lanjut tentang Variabel di bagian Python Variables.

Comments

Python memiliki kemampuan memberi komentar untuk tujuan dokumentasi dalam kode.

Komentar dimulai dengan "#", dan Python akan membuat sisa barisnya sebagai komentar:

Contoh

#Ini adalah komentar.
print ("Hello, World!")
  • Cobalah!
