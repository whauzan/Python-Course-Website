Python Comments

Komentar bisa digunakan untuk menjelaskan kode Python.

Komentar bisa digunakan untuk membuat kode lebih mudah dibaca.

Komentar bisa digunakan untuk mencegah eksekusi saat menguji kode.

Creating a Comment

Komentar diawali dengan #, dan Python akan menghiraukan sisa kode:

Contoh

#Ini adalah komentar.
print ("Hello, World!")
  • Cobalah!

Komentar dapat diletakkan di akhir baris kode, dan Python akan mengabaikan sisanya:

Contoh

print ("Hello, World!") #Ini adalah komentar.
  • Cobalah!

Komentar tidak selalu menjadi teks yang menjelaskan kode, tetapi juga bisa digunakan untuk mencegah Python untuk mengeksekusi kode:

Contoh

#print ("Hello, World!")
print ("Halo, Dunia!")
  • Cobalah!

Multi Line Comments

Python tidak benar-benar memiliki syntax untuk komentar multi-line.

Untuk menambahkan komentar multi-line kalian bisa menambahkan # untuk setiap baris.

Contoh

#Ini adalah komentar
#yang ditulis dalam
#lebih dari satu baris.
print ("Hello, World!")
  • Cobalah!

Atau, hal yang tidak disarankan, kalian bisa menggunakan multiline string.

Karena Python akan mengabaikan string yang tidak ditetapkan ke sebuah variabel, kalian dapat menambahkan multiline string (tanda kutip tiga) ''' dalam kode kalian, dan menempatkan komentar kalian di dalamnya:

Contoh

"""
Ini adalah komentar
yang ditulis dalam
lebih dari satu baris.
"""
print ("Hello, World!")
  • Cobalah!

Selama string tidak ditetapkan ke sebuah variabel, Python akan membaca kodenya, tapi kemudian mengabaikannya, dan kalian telah membuat komentar multiline.
