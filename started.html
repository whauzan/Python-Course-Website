Python Getting Started

Python Install

Kebanyakan PC atau Mac sudah memiliki Python yang sudah ter-install.

Untuk memeriksa apakah Python sudah ter-install di PC Windows, cari di Search bar "Python" atau jalankan perintah di bawah ini pada Command Line (cmd.exe):

C:\Users\ Nama User >python --version

Untuk memeriksa apakah Python sudah ter-install di Linux atau Mac, maka jika di Linux buka Command Line atau jika di Mac buka Terminal lalu jalankan perintah di bawah:

>python --version

Jika kalian tidak menemukan Python yang ter-install di komputer kalian, maka kalian bisa men-downloadnya secara gratis dari website https://www.python.org/.

Python Quickstart

Python adalah bahasa pemrograman yang diinterpretasikan, ini berarti bahwa sebagai developer kalian menulis file Python (.py) di teks editor dan kemudian memasukkan file-file itu ke dalam interpreter Python untuk dieksekusi.

Cara menjalankan file Python di command line adalah seperti ini:

C:\Users\ Nama User >python helloworld.py

Dimana "helloworld.py" adalah nama file Python kalian.

Mari kita coba membuat file Python pertama kita yang kita beri nama helloworld.py yang bisa dilakukan di teks editor manapun.

helloworld.py

print ("Hello, World!")
  • Cobalah!

Mudah bukan? Sekarang, simpan file "helloworld.py". Buka command line, arahkan ke directory tempat kalian menyimpan file, dan jalankan:

C:\Users\ Nama User >python helloworld.py

Output yang dihasilkan harusnya:

Hello, World!

Selamat! Kalian sudah menulis dan mengeksekusi program Python pertama kalian.

The Python Command Line

Untuk menguji kode yang pendek di Python terkadang akan lebih cepat dan lebih mudah untuk tidak menulis kode di dalam file. Hal ini dimungkinkan karena Python dapat dijalankan sebagai command line itu sendiri.

Ketik dan jalankan perintah berikut di Windows, Mac, atau Linux command line:

C:\Users\ Nama User >python

Atau, jika perintah "python" tidak berhasil, kalian bisa mencoba "py"

C:\Users\ Nama User >py

Dari situ kalian bisa menulis berbagai macam kode dalam Python, termasuk contoh hello world kita dari tutorial sebelumnya:

C:\Users\ Nama User >python
Python 3.6.4 (v3.6.4:d48eceb, Dec 19 2017, 06:04:45) [MSC v.1900 32 bit (Intel)] on win32
Type "help", "copyright", "credits" or "license" for more information.
>>> print ("Hello, World!")

Yang mana akan menghasilkan output "Hello, World!" di command line:

C:\Users\ Nama User >python
Python 3.6.4 (v3.6.4:d48eceb, Dec 19 2017, 06:04:45) [MSC v.1900 32 bit (Intel)] on win32
Type "help", "copyright", "credits" or "license" for more information.
>>> print ("Hello, World!")
Hello, World!

Setiap kali kalian selesai di command line Python, kalian cukup menulis kode di bawah ini untuk keluar dari interface command line Python:

exit()
