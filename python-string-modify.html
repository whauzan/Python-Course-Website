Python Modify Strings

Python memiliki seperangkat metode bawaan yang dapat kalian gunakan pada tipe data string.

Upper Case

Contoh

Metodeupper()mengembalikan string dalam huruf kapital:

a = "Hello, World!"
print (a.upper())
  • Cobalah!

Lower Case

Contoh

Metodelower()mengembalikan string dalam huruf kapital:

a = "Hello, World!"
print (a.lower())
  • Cobalah!

Remove Whitespace

Spasi putih adalah spasi sebelum dan/atau setelah teks sebenarnya, dan sering kali kalian ingin menghapus spasi ini.

Contoh

Metodestrip()menghapus spasi apa pun dari awal atau akhir:

a = " Hello, World! "
print (a.strip())    # mengembalikan "Hello, World!"
  • Cobalah!

Replace String

Contoh

Metodereplace()mengganti string dengan string lain:

a = "Hello, World!"
print (a.replace("H", "J"))
  • Cobalah!

Split String

Metodesplit()mengembalikan list di mana teks di antara pemisah yang ditentukan menjadi item list.

Contoh

Metodesplit()membagi string menjadi beberapa substring jika menemukan contoh pemisah:

a = "Hello, World!"
print (a.split(","))    # mengembalikan ['Hello', 'World!']
  • Cobalah!
